All Features · AB-SpectroX Gold

spectrum sweep · 27 waterfall palettes · power meter · IQ recorder · demodulation · markers · hardware control · bilingual UI

📡

Spectrum Sweep

  • 1 MHz – 7.25 GHz (HackRF)
  • 70 – 6000 MHz (Pluto extended)
  • RBW: 1k,2k,5k,10k,20k,50k,100k,200k,500k,1M,2M,5M Hz
  • FFT points: 8,192 – 262,144
  • Detectors: RMS, Average, Peak
  • Peak Hold (0-500ms fall rate)
  • Adaptive Spur Filter (learns 40MHz spurs per gain profile)

Waterfall / Spectrogram

  • 27 color palettes (Hot Iron, Viridis, Inferno, Plasma, Magma, Cividis, Turbo, Neon Aurora, Cyber Sunset, Deep Space Neon, Matrix Green, Ocean Depths, Sunset Glow, Military Amber, Medical Blue, Fire & Ice, Hacker Terminal, Professional Blue, Neon Pink, Electric Purple, Golden Hour, Arctic Ice, Lava Flow, Cyberpunk Neon, Royal Gold, Toxic Green, Deep Ocean)
  • Auto Palette mode (noise floor tracking)
  • Side colorbar with dBm scale
  • Mouse marker + double-click retune
  • Persistent display (1-60s decay)

Professional Power Meter

  • Units: dBm / Watt (full decimal)
  • External attenuator (0-120 dB)
  • Path loss (0-80 dB)
  • External offset (±40 dB)
  • Multi-point frequency calibration
  • Guided & Auto calibration wizard
  • Channel Power with ENBW (0.8-3.0)
  • SNR + Uncertainty estimation
  • JSON Save/Load
🎙️

Live Demodulation

  • FM (200kHz), AM (10kHz), LSB, USB, CW
  • Real-time IQ from HackRF/Pluto
  • DC removal, bandpass filters, de‑emphasis, AGC
  • Hiss reduction
  • PyAudio / sounddevice (48kHz, 16-bit)
  • Volume control + double-click retune
💾

IQ Recorder / Player / Tx

  • SigMF CS8 (Tx ready), C16, CF32
  • Duration / file size limits
  • SigMF metadata sidecar
  • Narrowband preview + auto-resume sweep
  • Playback with spectrum preview + loop
  • Transmit ci8 via HackRF/Pluto
🎯

Markers & UI

  • Max 8 markers (standard + peak)
  • Delta reference + custom colors + rename
  • Instant bilingual (EN/AR)
  • Show/hide modules (Waterfall, Demod, IQ, Power Meter)
  • Screenshot (PNG) + Save/Load all settings to JSON
  • Reset to defaults + capture folder selection

Advanced Hardware Control

  • HackRF One: LNA 0-40dB, VGA 0-62dB, LNA Amp (+14dB), Bias-T, Smart Gain Guard
  • ADALM-Pluto: RX gain 0-71dB manual + AGC (slow/fast), Auto RX gain suggestions, TX attenuation mapping
  • Start/Stop scan, Pause/Resume, center frequency, span, frequency range
  • FFT settings (Bin Width, Samples) + Power reference controls
  • Calibrated spectrum display + frequency range auto-detection (70-6000MHz extended)

Frequency Allocation Bands

  • Embedded Europe (ECA) & United States (FCC) tables
  • Semi-transparent colored overlays with descriptive tooltips
  • Auto-load at startup

Performance Optimizations

  • OpenGL acceleration (PyQtGraph) · Periodic garbage collection · Float32 spectrum · Adaptive FFT resize · High-priority Windows threading
