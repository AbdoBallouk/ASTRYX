مسح الطيف

  • مدى الترددات: 1 MHz إلى 7.25 GHz (HackRF) / 70 MHz إلى 6 GHz (Pluto)
  • عرض الحزمة اللحظي: 20 MHz (HackRF) / 56 MHz (Pluto)
  • خيارات RBW: 1k,2k,5k,10k,20k,50k,100k,200k,500k,1M,2M,5M Hz
  • نقاط FFT: 8,192 / 16,384 / 32,768 / 65,536 / 131,072 / 262,144
  • أنماط الكاشف: RMS، متوسط، ذروة (لقياس الاستطاعة)
  • تتبع القمم (Peak Hold) مع معدل هبوط 0-500 ms
  • مرشح توافقيات تكيفي: يتعلم التوافقيات الدورية (40MHz) ويحفظ الملفات الشخصية لكل ربح
  • تحديث طيف حي بمعدل ~20 إطار/ثانية · تسريع OpenGL
🌊

الشلال / المخطط الطيفي الزمني

  • 27 لوحة ألوان احترافية (Hot Iron, Viridis, Inferno, Plasma, Matrix, Cyberpunk, Ocean...)
  • وضع اللوحة التلقائي: ضبط ديناميكي للبداية/المدى بناءً على الإشارة وأرضية الضجيج
  • شريط ألوان جانبي بمقياس dBm
  • مؤشر الفأرة: خط أبيض مع قراءة التردد/الاستطاعة · نقر مزدوج لإعادة توجيه فك التعديل
  • قائمة النقر الأيمن: إضافة علامة، تعيين تردد التسجيل، بدء تسجيل IQ
  • عرض تراكمي (مخطط حراري) مع زمن اضمحلال 1-60 ثانية ولوحة ألوان منفصلة
⚡

مقياس الاستطاعة الاحترافي

  • الوحدات: dBm / واط (عرض عشري كامل)
  • تعويض المخفف الخارجي: 0 – 120 dB · خسارة الخط: 0 – 80 dB · الإزاحة الخارجية: ±40 dB
  • معايرة متعددة النقاط (تردد/إزاحة) مع استيفاء خطي
  • معالج معايرة إرشادي (قائمة ترددات مع الانتقال والمسح) ومعايرة آلية مع مولد sweep
  • حساب استطاعة القناة مع عامل ENBW (0.8-3.0 قابل للتعديل)
  • حساب SNR (ذروة ناقص المئين 20 للضجيج)
  • تقدير عدم اليقين · حفظ/تحميل المعايرة بصيغة JSON
  • أنماط القياس: استطاعة منخفضة، استطاعة متوسطة (قناة)، استطاعة عالية · كواشف: RMS، متوسط، ذروة
🎙️

فك التعديل الحي

  • الأنماط: FM (200 kHz), AM (10 kHz), LSB, USB, CW (1.2 kHz)
  • مصدر IQ لحظي من HackRF أو Pluto (معدل عينة 2 MHz لـ FM)
  • معالجة الصوت: إزالة DC، مرشحات تمرير النطاق، AGC، دي-إمفاسيس، تقليل الهسهسة
  • مخرج صوت: PyAudio أو sounddevice، 48 kHz، 16-bit
  • شريط تحكم بمستوى الصوت (0-100%) · إعادة توجيه بنقرة مزدوجة · إعادة ضبط الحالة تلقائياً
💾

مسجل IQ / مشغل / مرسل

  • صيغ التسجيل: SigMF CS8 (جاهز للإرسال)، SigMF C16، Complex Float32 (.cf32)
  • حد أقصى للمدة/حجم الملف · كتابة بيانات وصفية (SigMF sidecar)
  • معاينة ضيقة أثناء التسجيل · استئناف المسح تلقائياً بعد التسجيل
  • إحصائيات لحظية (المدة، الحجم، عمق الطابور)
  • إعادة التشغيل: فتح ملف مع معاينة الطيف · وضع التكرار
  • إرسال (Tx): ملفات ci8 عبر HackRF/Pluto مع ربح قابل للضبط
🎯

العلامات وواجهة المستخدم

  • حد أقصى 8 علامات (قياسية + ذروة) · مرجع دلتا · ألوان قابلة للتخصيص · إعادة تسمية
  • إضافة علامة يدوية/ذروة من قائمة النقر الأيمن · حذف بزر "حذف" أو مفتاح Delete
  • واجهة ثنائية اللغة فورية (عربي/إنجليزي)
  • إظهار/إخفاء الوحدات (شلال، فك تعديل، مسجل IQ، مقياس استطاعة)
  • شريط معلومات التردد · زر إخفاء اللوحة الجانبية
  • لقطة شاشة (PNG) للطيف والشلال معاً
  • حفظ/تحميل جميع الإعدادات إلى ملف JSON واحد · إعادة الضبط الافتراضي · تحديد مجلد الالتقاط

التحكم المتقدم بالعتاد

  • HackRF One: ربح LNA 0-40dB، ربح VGA 0-62dB، مضخم LNA (+14dB)، Bias-T (14V)، حماية الربح الذكية (تخفيض تلقائي لـ VGA)
  • ADALM-Pluto: ربح استقبال يدوي 0-71dB + أوضاع AGC (يدوي، بطيء، سريع)، اقتراح ربح استقبال تلقائي حسب التردد/المدى، تعيين تخفيف الإرسال
  • قائمة منسدلة للمنصة مع إعادة تشغيل تلقائي · أزرار بدء/إيقاف المسح، إيقاف مؤقت/استئناف
  • عناصر تحكم بالتردد المركزي، المدى، الترددات (بداية/نهاية) · إعدادات FFT
  • عناصر تحكم مرجع الاستطاعة · عرض طيف معاير · كشف تلقائي لمدى التردد لـ Pluto

نطاقات تخصيص التردد

  • جداول تخصيص التردد الأوروبية (ECA) والأمريكية (FCC) محملة من CSV
  • تراكبات شبه شفافة ملونة على مخطط الطيف مع تلميحات باسم النطاق والتطبيقات · تحميل تلقائي عند بدء التشغيل

تحسينات الأداء

  • تسريع OpenGL عبر PyQtGraph
  • جمع القمامة الدوري (كل 50 إطاراً) · تقليل حجم البيانات (float32)
  • تكيف FFT · خيوط Windows ذات الأولوية العالية · تحديث محدود الإطارات (~20 fps)

ملاحظات المستخدم والإشعارات

  • شريط الحالة مع رسائل الخطأ · شارة وقت التشغيل (IQ REC, TX LIVE, DEMOD, FILE PREVIEW)
  • إشعارات Pluto في شريط الحالة ولوحة الإعدادات · مربعات حوار خطأ للمشاكل الحرجة
