academic & research value

An Integrated Framework for Academic Education & Advanced Research

Bridging the gap between expensive commercial analyzers and complex open-source tools

This professional software and engineering platform was purpose-built to eliminate the historical gap separating prohibitively expensive traditional laboratory hardware analyzers from complex open-source command-line tools that lack cohesive graphical frameworks. The system introduces an advanced, interactive visual environment enabling engineers, researchers, and university faculties to monitor and inspect live RF signals from 1 MHz to 7.25 GHz in real time with high mathematical precision. It yields instantaneous parametric extraction of critical signal profiles, including absolute power density, instantaneous occupied bandwidth, and Signal-to-Noise Ratio (SNR).

Spectrum Sweep

  • Frequency range: 1 MHz – 7.25 GHz (HackRF One)

  • Instantaneous bandwidth: Up to 20 MHz (HackRF) / 56 MHz (Pluto)
  • RBW options: 1k, 2k, 5k, 10k, 20k, 50k, 100k, 200k, 500k, 1M, 2M, 5M Hz
  • FFT points: 8,192 / 16,384 / 32,768 / 65,536 / 131,072 / 262,144
  • Detector modes: RMS, Average, Peak (for Power Meter)
  • Peak Hold with adjustable fall rate: 0-500 milliseconds
  • Adaptive Spur Filter: learns 40MHz periodic spurs per gain profile
  • Real-time spectrum update at ~20 FPS
  • OpenGL hardware acceleration via PyQtGraph

Professional Power Meter

  • Units: dBm / Watt (full decimal display)
  • External attenuator compensation: 0 – 120 dB
  • Path loss compensation: 0 – 80 dB
  • External offset: ±40 dB
  • Multi-point frequency calibration: list of (freq, offset) with linear interpolation
  • Guided calibration wizard: builds frequency list, steps through each frequency with capture prompt
  • Auto calibration: works with sweeping signal generator, tracks peaks within tolerance window
  • Channel Power calculation with ENBW factor (0.8-3.0 adjustable, default 1.06)
  • SNR calculation: peak minus noise floor (20th percentile)
  • Uncertainty estimation based on variance, attenuation, SNR, and signal level
  • Save/Load calibration to JSON format (points, ENBW, losses, offset, attenuator used)
  • Measurement modes: Low Power, Average (Channel Power), High Power
  • Detectors: RMS, Average, Peak

Live Demodulation

  • FM mode: 200 kHz bandwidth with de‑emphasis and channelized filtering
  • AM mode: up to 10 kHz bandwidth with linear envelope extraction
  • LSB / USB: Single Sideband processing (1 ± Q analytical signals)
  • CW mode: 1.2 kHz bandwidth for morse/telemetry isolation
  • Real-time IQ source from HackRF or Pluto (2 MHz sample rate for FM, adjustable for other modes)
  • Audio processing: DC removal, bandpass filters, AGC, hiss reduction
  • Audio output: PyAudio or sounddevice, 48 kHz, 16-bit integer
  • Volume control slider: 0 – 100%
  • Retune by double-click on spectrum or waterfall
  • State reset on frequency or mode change (internal filters reset automatically)
  • Independent live spectrum mini-display for downconverted passband

  • Power reference controls: Ref Level (dBm) + Step (dB) + Apply + Auto

Performance Optimizations

  • OpenGL acceleration via PyQtGraph (enabled by default)
  • Periodic garbage collection: every 50 frames in main thread

  • Adaptive FFT resize: auto-resizes waterfall width based on FFT points
  • High-priority Windows threading for real-time performance
  • Frame-limited update: spectrum and waterfall updated at ~20 fps
