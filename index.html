All Features · ASTRYX Gold Edition

spectrum sweep · waterfall (27 palettes) · power meter · IQ recorder · demodulation · markers · hardware control · bilingual UI

Spectrum Sweep

  • 📡 1 MHz – 7.25 GHz (HackRF) / 70–6000 MHz (Pluto extended)
  • ⚡ RBW: 1k,2k,5k,10k,20k,50k,100k,200k,500k,1M,2M,5M Hz
  • 🎛️ FFT points: 8192, 16384, 32768, 65536, 131072, 262144
  • 📊 Detectors: RMS, Average, Peak (Power Meter)
  • 📈 Peak Hold with adjustable fall rate 0-500 ms
  • 🛡️ Adaptive Spur Filter: learns 40MHz periodic spurs per gain profile

Waterfall Spectrogram

  • 🎨 27 color palettes (Hot Iron, Viridis, Inferno, Plasma, Matrix, Cyberpunk, Ocean, Military Amber, Medical Blue, etc)
  • 🤖 Auto Palette mode: adjusts start/range based on signal/noise
  • 📏 Side colorbar with dBm scale
  • 🖱️ Mouse marker: white dashed line + freq/power readout
  • 🔁 Double‑click to retune demodulator
  • ⏱️ Persistent display with adjustable decay (1-60s)

Professional Power Meter

  • 📏 Units: dBm / Watt (full decimal)
  • 🔧 External attenuator compensation (0-120 dB) + Path loss (0-80 dB) + External offset (±40 dB)
  • 📊 Multi‑point frequency calibration (list, linear interpolation, apply range)
  • 🧪 Guided calibration: steps through frequencies with capture prompts
  • 🤖 Auto calibration: works with sweeping signal generator, peak tracking
  • 📐 Channel Power with ENBW factor (0.8-3.0 adjustable)
  • 📉 SNR calculation (peak minus 20th percentile noise)
  • ⚠️ Uncertainty estimation based on variance, attenuation, SNR
  • 💾 Save/Load calibration JSON (points, ENBW, losses, offset, attenuator)

Live Demodulation

  • 🎙️ Modes: FM (200kHz), AM (10kHz), LSB, USB, CW (1.2kHz)
  • 📡 IQ source: real-time HackRF/Pluto (2MHz sample rate for FM)
  • 🎛️ Audio processing: DC removal, bandpass filters, de‑emphasis (FM), AGC, hiss reduction
  • 🔊 Output: PyAudio / sounddevice, 48kHz, 16‑bit int
  • 🔁 Retune by double‑click on spectrum/waterfall
  • 🔄 State reset on frequency/mode change

IQ Recorder / Player / Transmitter

  • 💾 SigMF CS8 (.sigmf-data) – Tx ready, SigMF C16, Complex Float32 (.cf32)
  • ⏺️ Recording: max duration, max file size, SigMF metadata sidecar, narrowband preview, auto-resume sweep
  • ▶️ Playback: open .sigmf-data/.cf32, spectrum preview, loop mode
  • 📡 Transmit (Tx): ci8 files via HackRF/Pluto, adjustable frequency and gain (0-47dB HackRF, mapped attenuation for Pluto)

Markers & UI

  • 📍 Max 8 markers: standard and peak (solid above, dashed below)
  • 📐 Delta reference, customizable colors, rename, add from context menu or peak cycling
  • 🌍 Instant bilingual (EN/AR) – all strings, tabs, buttons
  • ⚙️ Show/hide modules: Waterfall, Demodulator, IQ Recorder, Power Meter
  • 📸 Screenshot captures spectrum + waterfall together as PNG
  • 💾 Save/Load all settings (device, display, demod, IQ, markers, calibration, spur profiles) to single JSON
  • 🔄 Reset to defaults

Advanced Hardware Control

  • 🎛️ HackRF: LNA 0-40dB, VGA 0-62dB, LNA Amp (+14dB), Bias‑T, Smart Gain Guard (auto VGA reduce when Amp on)
  • 🎛️ Pluto: RX gain 0-71dB manual, AGC modes (slow/fast), Auto RX gain suggestions based on freq/span, TX attenuation mapping
  • 📡 Frequency range auto‑detection (Pluto probes RX limits, 70-6000MHz extended)
  • 🔄 Start/Stop scan, Pause/Resume, center frequency spinbox, span spinbox

Allocation Bands & Performance

  • 🗺️ Europe / USA frequency allocation bands (semi‑transparent overlays with tooltips)
  • ⚡ Performance: OpenGL in PyQtGraph, periodic garbage collection, float32 spectrum, adaptive FFT resize
  • 📊 Runtime badge (IQ REC, TX LIVE, DEMOD, FILE PREVIEW)
  • 📁 Capture folder selection for screenshots and IQ recordings
