Markers & User Interface

  • Max 8 markers (standard + peak)
  • Delta reference marker
  • Customizable colors
  • Rename markers
  • Add from context menu
  • Peak cycling (add peak marker)
  • Instant bilingual (EN/AR)

Advanced Hardware Control

  • HackRF One: LNA 0-40dB, VGA 0-62dB, LNA Amp (+14dB), Bias-T, Smart Gain Guard (auto VGA reduce when Amp on)
  • ADALM-Pluto: RX gain 0-71dB manual, AGC modes (slow/fast), Auto RX gain suggestions based on freq/span, TX attenuation mapping
  • Frequency range auto‑detection (Pluto probes RX limits, 70-6000MHz extended)
  • Start/Stop scan, Pause/Resume, center frequency spinbox, span spinbox

Frequency Allocation Bands

  • Embedded Europe (ECA) frequency allocation tables
  • Embedded United States (FCC) frequency allocation tables
  • Semi-transparent colored overlays on spectrum plot
  • Descriptive tooltips with band name and applications
  • Auto-load at startup
